# Módulo 1 — Análise de Perfil

## Objetivo
Extrair inteligência estratégica completa de prints de perfil do Instagram para
construir uma "persona de conteúdo" que guia toda a geração posterior.

---

## Checklist de Extração Visual

### 1. Bio Screen
- [ ] @usuário e nome de exibição → identidade e SEO do perfil
- [ ] Categoria (criador, negócio, pessoal) → contexto institucional
- [ ] Headline da bio → proposta de valor declarada
- [ ] Bullet points / linhas da bio → sub-nichos e diferenciais
- [ ] Emojis → personalidade e energia da marca
- [ ] Link na bio → oferta principal (produto, curso, WhatsApp, site)
- [ ] Botão de ação (mensagem, e-mail, telefone) → intenção de conversão
- [ ] Número de seguidores / seguindo / posts → maturidade do perfil

### 2. Feed Grid (9-12 primeiros posts)
- [ ] Cores dominantes → identidade visual (paleta)
- [ ] Presença de rosto humano → conexão emocional
- [ ] Proporção Reel vs Foto vs Carrossel → estratégia de formato
- [ ] Consistência de estética → profissionalismo percebido
- [ ] Temas recorrentes → pilares de conteúdo
- [ ] Qualidade de produção → nível de investimento
- [ ] Posts fixados (pinned) → o que o perfil quer que você veja primeiro

### 3. Métricas visíveis
- [ ] Visualizações de Reels → alcance real
- [ ] Curtidas → engajamento de aprovação
- [ ] Comentários → engajamento de conexão
- [ ] Proporção views/seguidores → eficiência do conteúdo

### 4. Destaques (Highlights)
- [ ] Quantidade de destaques → organização do perfil
- [ ] Títulos dos destaques → categorias de conteúdo arquivado
- [ ] Capas dos destaques → identidade visual dos Stories

---

## Output da Análise

Gere o diagnóstico no formato definido no SKILL.md principal (seção "Formato de Entrega").

### Classificação de Maturidade do Perfil:

| Fase | Seguidores | Engajamento médio | Estratégia recomendada |
|------|-----------|-------------------|----------------------|
| Iniciante | < 1k | qualquer | Consistência + Comunidade |
| Crescimento | 1k-10k | >3% | Viral + Frequência |
| Consolidação | 10k-50k | >2% | Autoridade + Conversão |
| Referência | 50k-500k | >1% | Monetização + Parcerias |
| Grande | 500k+ | >0.5% | Escala + Marca |

### Tom de Voz — Matriz de Detecção:

Analisar legendas e comunicação visual para classificar em:

- **Educador** → ensina, explica, simplifica
- **Inspirador** → motiva, emociona, transforma
- **Entertainer** → diverte, provoca, surpreende
- **Autoridade** → dados, cases, resultados
- **Amigo** → conversa, vulnerável, próximo
- **Vendedor** → oferta clara, urgência, benefício

O perfil pode ter combinações. Identificar o dominante e o secundário.

## Ficha de Análise de Reel/Post

Quando o usuário enviar link, print ou descrever um conteúdo existente, preencher:

NICHO: _______________________________________________
FORMATO: [ ] Reel  [ ] Carrossel  [ ] Post único  [ ] Story
TEMA CENTRAL: ________________________________________

GANCHO (primeiros 3 segundos):
  Tipo: [ ] Pergunta  [ ] Afirmação impactante  [ ] Dado  [ ] Visual surpresa
  Conteúdo: ___________________________________________

ESTRUTURA DE CENAS:
  Cena 1: _____________________________________________
  Cena 2: _____________________________________________
  Cena 3: _____________________________________________

ELEMENTOS VISUAIS:
  Apresentador: [ ] Pessoa real  [ ] Avatar/AI  [ ] Sem pessoa
  Estilo visual: [ ] Profissional  [ ] Casual  [ ] Lifestyle  [ ] Técnico

LEGENDA/CTA identificado: _____________________________

PONTOS FORTES:
  1. __________________________________________________
  2. __________________________________________________

PONTOS A MELHORAR:
  1. __________________________________________________
  2. __________________________________________________
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

### Quando o link do Instagram está bloqueado (situação comum):
Dizer ao usuário:
> "O Instagram restringe acesso direto. Me conta: qual é o tema do reel/post? O que aparece nas cenas? Tem narração? Qual é a legenda? Com isso, crio o pacote completo."

## Perguntas de Confirmação (após análise)

Sempre confirmar com o usuário antes de gerar conteúdo:

1. "Confirmando: seu nicho principal é [X] com foco em [Y]. Está correto?"
2. "Qual é o seu principal objetivo agora: mais engajamento, mais seguidores ou mais conversão?"
3. "Tem algum formato de conteúdo que você prefere criar (Reel, Carrossel, Feed)?"
4. "Quer que eu mantenha o tom [detectado] ou prefere um tom diferente?"
