# Módulo 5 — Pacote de Post (Feed + Carrossel + Stories)

## Estrutura de Legenda

```
[GANCHO — linha 1]
Frase que PARA o scroll. Pergunta, dado, afirmação ousada.

[ESPAÇO EM BRANCO — estratégico]

[CORPO — 2-3 parágrafos curtos]
Linha 1 do parágrafo.
Linha 2 — sem paredes de texto.

→ Use símbolos para facilitar leitura
→ Emojis estratégicos (não spam)

[CTA]
Uma ação. Direta. Simples.
👇 Comenta aqui / Salva esse post / Me manda DM

[HASHTAGS]
```

## Carrossel — Estrutura de Slides

| Slide | Função | Regra |
|-------|--------|-------|
| 1 | Capa / Gancho | Título + promessa visual. Deve funcionar sozinho. |
| 2 | Introdução | Contexto rápido. Por que isso importa? |
| 3-7 | Conteúdo | 1 ideia por slide. Curto. Visual. |
| 8-9 | Síntese | Resumo ou checklist salvável |
| 10 | CTA | Ação + pergunta nos comentários |

### Regras visuais do carrossel:
- Elemento "continuação" na borda direita do slide → incentiva swipe
- Cor de fundo consistente em todos os slides
- Tipografia legível em tela pequena
- Máx 40 palavras por slide

## Stories de Apoio

Para cada post, gerar 2-3 stories complementares:

1. **Story de Tráfego** → "Novo post no feed! Já viu?" + [ação: ver post]
2. **Story de Engajamento** → Poll ou pergunta relacionada ao tema
3. **Story de Conversão** → CTA direto para produto/serviço/link

## Variações de Conteúdo

Após entregar o pacote principal, sempre oferecer 2-3 variações:

```
🔄 VARIAÇÕES — [TEMA]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

💡 VARIAÇÃO 1 — [TÍTULO]
Ângulo: [mudança de perspectiva]
Gancho: "[texto dos primeiros 3 segundos]"
Estrutura: [Cena 1] → [Cena 2] → [CTA]
Formato ideal: [ ] Reel  [ ] Carrossel  [ ] Story
Diferencial: [por que funciona diferente do original]

---

💡 VARIAÇÃO 2 — [TÍTULO]
Ângulo: [mudança de formato]
Gancho: "[...]"
Estrutura: [...]
Formato ideal: [...]
Diferencial: [...]

---

💡 VARIAÇÃO 3 — [TÍTULO] (Talking Object / Trend)
Ângulo: [formato trending ou objeto falante]
Gancho: "[...]"
Estrutura: [...]
Formato ideal: [...]
Diferencial: [...]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

### Frameworks de variação:
1. **Mudança de ângulo** — mesmo tema, perspectiva oposta
2. **Mudança de formato** — Reel narrativo → Carrossel / Tour → Antes e Depois
3. **Série** — transforma 1 post em 3-5 partes
4. **Trend adaptada** — POV, Expectativa x Realidade, Tipos de [X]
