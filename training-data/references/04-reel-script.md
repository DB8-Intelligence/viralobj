# Módulo 4 — Roteiro de Reel

## Estrutura Universal de Reel Viral

```
[0-2s]   HOOK → Para o scroll imediatamente
[2-5s]   PROMESSA → O que a pessoa vai aprender/ver/sentir
[5-30s]  ENTREGA → Conteúdo de valor (educação, transformação, humor)
[30-35s] CONCLUSÃO → Síntese poderosa ou virada
[35-38s] CTA → Uma ação clara e simples
```

## Regras de Roteiro

### Narração de Voz:
- [pausa 0.5s] → respira entre tópicos
- [pausa longa 1s] → antes de revelação ou dado importante
- [ÊNFASE] → palavra que deve ter entonação forte
- [velocidade lenta] → para impacto em momentos-chave
- Tempo médio: 2.5 palavras por segundo
- Tom conversacional, não "locutor de rádio"

### Texto na Tela:
- Máx 6 palavras por frame
- Fonte grande, legível em celular pequeno
- Contraste alto (branco no escuro, preto no claro)
- Aparece sincronizado com a voz
- Pode ter emoji estratégico (máx 1 por linha)

### Prompts de Imagem AI:
Sempre em inglês. Estrutura:
```
[art style], [main subject], [action/pose], [environment/background],
[lighting: golden hour/studio/neon/natural], [mood: dramatic/warm/mysterious/energetic],
[camera: close-up/wide shot/overhead], [quality: cinematic, 8K, hyperrealistic],
aspect ratio 9:16
```

## Variações de Duração

| Duração | Uso ideal | Taxa de conclusão esperada |
|---------|-----------|---------------------------|
| 7-15s | Viral máximo, hook puro | 80-90% |
| 15-30s | Educativo rápido | 60-75% |
| 30-60s | Tutorial, lista | 40-60% |
| 60-90s | Story completo, case | 30-45% |

## Consistência de Personagem Entre Cenas

Para manter o mesmo personagem AI em múltiplas cenas:

```
Base do personagem (repetir em TODOS os prompts da série):
"[Nome]: Brazilian [gender], [age range], [hair description], [distinctive feature],
[typical outfit] — maintain consistent appearance across all scenes"
```

## Checklist Pré-Entrega

- [ ] Hook nos primeiros 2 segundos é irresistível?
- [ ] A promessa do hook é cumprida dentro do reel?
- [ ] Texto em tela sincronizado com voz?
- [ ] CTA claro e único?
- [ ] Sem intro longa ("Oi gente, hoje eu vou...")?
- [ ] Prompts de imagem em inglês com aspect ratio 9:16?
- [ ] Ferramenta recomendada indicada?
- [ ] Variações oferecidas ao final?
- [ ] **PRO MAX:** Pronto para auto-post via Meta API?
